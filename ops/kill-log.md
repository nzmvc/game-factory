## Kayıt şablonu

```markdown
## <Oyun adı> — ÖLDÜRÜLDÜ (YYYY-AA-GG)

- **Aşama:** hangi makinede öldü (M2 seçim / M6 test / M7 veri)
- **Harcanan:** efor (gün) + para ($)
- **Veriler:** CPI $X · CTR %X · D1 %X · playtime X dk (eşik karşılaştırmalı)
- **Neden öldü:** eşik tablosuna göre tek cümle
- **Ne öğrendik:** 1–3 madde, tekrar kullanılabilir biçimde
- **Ne taşınır:** başka adaya aktarılabilir parça (mekanik, kreatif içgörüsü, kod modülü)
- **Karar kaydı:** DECISIONS.md satırına link
```
