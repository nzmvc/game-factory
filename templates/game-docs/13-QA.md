## 1. "Bitti" tanımı (Faz 1 prototip)

Temiz kurulumdan **10 dakika kesintisiz oynanış**: crash yok, akış tıkanmıyor,
juice hissi yerinde, event'ler doğru akıyor.

## 2. 10 dakika oynanış çeklisti

- [ ] Seviye akışı: Win → next → Win zinciri <5+> seviye sorunsuz
- [ ] Fail akışı: fail → rewarded devam → devam ediyor; Retry çalışıyor
- [ ] Near-miss anı hissediliyor (UI gerilimi + ses)
- [ ] Kombo juice zinciri (pitch artışı) çalışıyor
- [ ] FPS stabil; input gecikmesi yok
- [ ] Booster akışları (rewarded stub) çalışıyor
- [ ] Ayarlar: ses/haptik kapatma anında etkili
- [ ] İlerleme kaydı: uygulamayı öldür → aç → kaldığın yerden

## 3. Analytics doğrulaması

- [ ] Şemadaki her event en az bir kez tetiklenip loglandı (şasi-lite: konsol;
      full: Firebase DebugView)
- [ ] Parametre değerleri anlamlı (duration > 0, level_id doğru)
